### ☁️ Efficient & Secure File Distribution
A robust file-sharing platform designed for seamless uploads and instant sharing. By leveraging **Supabase Storage** and **PostgreSQL**, this app ensures your files are stored securely and accessible from anywhere with a generated share link.

### ✨ Platform Features
- 📤 **Instant Upload:** Drag-and-drop interface for fast file processing.
- 🔗 **Shareable Links:** Generate public or private links for your assets.
- 🔐 **User Management:** Secure dashboard to view, delete, and manage your files.
- 💾 **Supabase Integration:** High-performance storage with Row Level Security (RLS).
- 📱 **Mobile Optimized:** Upload and share files on the go with a fully responsive UI.

### 🛠️ Technical Stack

| Feature | Technology | Purpose |
| :--- | :--- | :--- |
| **Frontend** | React + Vite | Fast UI rendering and state management |
| **Storage** | Supabase Storage | S3-compatible bucket for file hosting |
| **Database** | PostgreSQL | Storing file metadata and user relationships |
| **Auth** | Supabase Auth | Protected routes and user-specific folders |
| **Styling** | Tailwind CSS | Clean, modern, and high-contrast design |
